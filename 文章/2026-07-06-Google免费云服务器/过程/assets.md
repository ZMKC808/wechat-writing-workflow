# 已复制图片

![[images/01-region-us-central1.jpg]]

用途：区域选择。文案旁边强调免费区域只有 `us-west1`、`us-central1`、`us-east1`。

![[images/02-e2-micro.jpg]]

用途：机器配置。文案旁边强调 Series 选 `E2`，Machine type 选 `e2-micro`。

![[images/03-standard-persistent-disk.jpg]]

用途：磁盘配置。文案旁边强调 Boot disk type 选 `Standard persistent disk`，不要选 Balanced / SSD。

![[images/04-standard-network-tier.jpg]]

用途：网络配置。文案旁边强调 Network Service Tier 选 `Standard`，否则 Premium 默认更容易产生出站流量费用。

# 建议补图

- Data Protection 页面：`No backups`。
- Observability 页面：如有 `Install Ops Agent`，建议关闭。
- 最终创建前右侧费用估算页。
- 第二天 Billing / Reports 页面，证明配置无误或定位小额扣费来源。
