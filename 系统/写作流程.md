# 公众号写作最短路径

这是给**你**看的流程概览，具体执行细节交给 Codex 按 WORKFLOW.md 跑。

## 1. 启动

给 Codex 发链接/图片/热点，说"帮我写"。

## 2. 产出

Codex 会在 `文章/` 下建项目文件夹，里面包含：

- `文章.md` — 正文稿
- `发布版.md` — 发布格式
- `图片/` — 配图
- `过程/` — 调研材料

## 3. 终审

你看一遍正文，没问题就发布。

## 4. 发布后

告诉 Codex 已发布，它会读阅读数记到 `复盘/`。
