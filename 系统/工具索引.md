## 常用命令

- `agent-reach doctor --json`：检查全网调研通道。
- `camofox-health`：检查公众号 / 浏览器读取服务。
- `cd ~/tools/MediaCrawler && uv run python main.py`：运行 MediaCrawler。
- `wechat-article-to-markdown URL -o DIR`：微信公众号文章转 Markdown。
- `agent-tools-doctor`：检查本机 agent 工具环境。
