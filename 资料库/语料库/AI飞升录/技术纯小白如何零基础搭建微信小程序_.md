项目背景

我是**文科生** 并没有技术背景 本次能搭建成功归功于原作者详细的**教程**和万能的**KIMI**

项目开发周期约**2周**（含备案等待）项目总搭建成本约**80元** （56服务器1域名 20微信认证 3解析/月）

本菜鸟**第一次**写技术教程 多有**遗漏错误** 欢迎指正！！

最初我是因为 之前付费的去水印小程序无法打开 刚好需要用到 发现网上去水印工具都有**大量广告**和**付费入口**，所以干脆想着做一个方便给**“胶己人”**用的小程序工具。

于是 我通过**谷歌搜索**找到 \[2023去水印小程序源码 前端+后端 独立后台 小白亲测可用\] （https://blog.csdn.net/chuanznn/article/details/130579992）

并且从👆**CSDN文章**里挖到了原作者的搭建教程 \[橘猫去水印搭建教程\]

（https://www.kancloud.cn/xiaofanmo/jumao\_free/3113503）

按照教程，我开始下载并安装**本地的工具包＆资料包**，并全部放在本地的文件夹**“它喵小程序搭建”**里等待备用。

**连接**堡塔SSH终端****

1.打开前面下载好的宝塔SSH终端（https://link.zhihu.com/?target=https%3A//www.bt.cn/new/product\_ssh.html）

3.端口：22（Linux系统默认为22，如果是其他端口的话，购买的服务器管理面板一般有标注）

4.SSH账号：root（linux系统默认为root，如果是其他端口的话，购买的服务器管理面板一般有标注）

5.保存并连接

为确保账号运营主体经营状态存续、命名合规、行业资质真实有效并且在线提供的服务正常访问。

**有效时间**

参考微信小程序企业认证，认证有效期是一年，因此一年后开发者必须重新缴费完成小程序认证。

### 科普的技术问题

**1.什么是API？**

别人写好的代码，或编译好的程序，提供给你使用，就叫做api。常称 接口。

**2.为什么要设置合法域名？**

因为小程序后台要调用api（接口）用来访问 下载 展示等\*\*

**3.合法域名的作用＆条件**

**作用**

1.作为小程序请求的 API 服务器域名，包括 HTTP API、WebSocket 等。

2.作为小程序下载或展示资源的服务器域名，包括上传图片和下载图片等。

3.作为小程序 WebSocket 信道的服务器域名。

**条件**

1.必须是已备案的域名或者 IP 地址。

2.必须采用 HTTPS 协议，并且要求 SSL 证书必须是由权威机构颁发的有效证书。

3.需要支持 TLS1.2 及以上协议。

4.不支持 IP 地址直接绑定域名。

5.不支持端口号的形式，如www.example.com: 8080。

6.不支持泛域名（如：\*.example.com）。

**4.网站备案名称要求和注意项?**

**网站名称的基本要求**

非国家级单位网站名称不能包含 “中国”、“中华”、“国家”、“人名”、“地名” 等字样。

网站名称不能是纯数字、纯英文、不能包含特殊符号和敏感词汇；个人网站名称不能以域名、姓名命名，不能包含公司、组织等类型的字眼。

网站名称要能体现网站的内容。

网站名称要求三个字以上（含三个字）。

**个人网站名称**

1.个人名义主办的网站类型主要有：工作经验分享（技术、学术研究等）、摄影作品分享、个人生活感悟记录等，网站名称要尽量贴近网站内容。

说明：以下示例可供参考，请勿抄袭下方网站名称，名称使用过多会被列入通信管理局黑名单。

工作经验分享类网站：网页制作经验谈、小工具应用、XXX 的研发日志、前端经验分享、自动化测试开源、文字极客、个人开发测试使用网站、PHP 开发等。

摄影作品分享：我的照片墙、雕刻时光、我的摄影小空间、我的作品展览等。

个人生活感悟：人生百味、吃货分享、美食分享、快乐小窝、柠檬草的味道、一起考研吧、怀念时光、时光飞逝、电影爱好小分享、个人杂货铺、我的生活秀等。

**企业网站名称**

推荐使用公司全称或简称作为网站名称，允许多个网站的名称相同。

**网站名称的建议**

网站名称所有管局总体要求均为必须和主体性质相符，然而每个省的审核标准并不完全一致，以下几点建议供您参考：

主办单位不是国家级单位，其网站不能以 “中国”、“国家” 等关键词命名，如果含有必须提供相关职能部门的审批文件。

个人备案，网站名称不能涉及到 “某公司”、“某俱乐部”、“某团体”、“某社区” 等明显非个人性质的网站名称。

网站名称应与前置审批或专项审批的取得情况相对应。无新闻、出版、药监、文化、广电等前置审批或专项审批的网站，不能以相关领域关键词命名。

若网站名称或服务内容涉及到金融类，则需尽可能在网站名称上规避关键词。若确实涉及，建议先到本省金融办进行办理专项审批。提交时，同时上传审批文件电子版，否则有被退回的风险。

个人备案的网站名称，不允许涉及经营性内容，如 “购买”、“买卖”、“交易”、“优惠券”、“电子商务” 等。

淘宝客等返利性质的网站（通过这个网站去购物，购物结束后获取返利），一般管局都不要求进行经营性备案申请，但是若网站名称涉及电商类敏感关键字将会增加退回风险

使用与企业经营范围完全不相关的产品来命名，一般都比较难通过。可在备注增加相关说明，以增加通过的可能性。

个人网站名称不能包含地名；单位备案时，主体名称中包含的地名要大于或者等于网站名称中包含的地域。

个人网站备案，网站名称若含有人名，不能为个人姓名之外的人名。例如，张三申请备案，网站叫李四的个人网站；一般笔名、网名等明显是人名又和证件上对应不上的，都会有退回风险，命名时建议尽可能规避。

“个人空间”、“个人博客” 等网站名称，需尽可能规避，因为这样的名称过于宽泛不具体。建议使用网站的主要内容作为网站名称。

企业网站名称，并不要求只能为 “公司名称” 或“公司简称”，针对这类网站名称的通过率比较高，建议采用。

项目总结  

当我们用旁观者的视角时，技术啊乱七八糟啊总是诡谲神秘的。（当我回顾前文时，我也被吓一跳，因为看起来确实有点“专业”） 但我很清楚这只是执行过程的正常”信息”，看似复杂而已。

如果你对技术一窍不通却跟我一样十分感兴趣 渴望更多实战中能使用上“技术”的小微个体户 欢迎你来找我们交流，凑满10人我会拉个小群。

说回来我实战过程中，发现市面上很多所谓“小程序搭建”，就是套壳行为，技术门槛极低 利润空间极高。

让我印象深刻的是我一位网络朋友，仅仅小学学历且家境艰难就因为接触互联网比较久，现在靠着给各种老板和个体代搭建网站等都能每个月轻松赚到几万块（感兴趣可以找我细聊这个案例。）

对此，我基于这次技术实战成功给出一个大胆推论 即“只要我能拥有绿色完整的小程序源码我就能搭建任何市场上已有的任意微信小程序” 因为“搭建”的底层逻辑是相通的，且我现在拥有了强大而耐心的AI技术顾问，等有空我还会再验证一下。

此外，微信小程序创业对普通人来说，依旧是很好的低成本创业机会，特别是从事微商网赚行当的群体，适当的技术加持就能实现很好的效果。

比如搭建一个网赚代理商认证系统 就能给人一种更为专业更为大户做派的质感。

比如搭建一个工具类小程序通过做排名群推等来获取流量主收益，只要切中需求点，就会比开实体门店更轻也更有机会。（给实体老板说下：这相当于你在微信用超低的成本租了个带功能的店铺）

另外，涉及到企业如何大额的节省服务器费用、如何申请互联网信息经营许可证（ICP） 如何悬挂备案号、如何找到合适的源码、如何找到超炸的技术实战归档...都属于市场的“技术服务＆信息差”，如果你没有钻研精神没有正确打开方式，那你就会像无头到处苍蝇乱撞，最终败兴而归。

直接付费确实能提供代办理，但多数时候技术是一个系统性问题（很难一次性解决）。如果你想“不求人” 那你就需要招聘技术专员，而这对于小白来说显然不够友好。

所以你如果你有搭建同款的想法或搭建方面的问题，欢迎任意打赏后过来咨询我吧~
