龙虾之父都在打听的中国神秘模型？居然是…
====================

原创 阿竹 AI飞升录 2026-03-19 18:49 浙江

> 原文地址: [https://mp.weixin.qq.com/s/t1Tta6pdFNqIHkwU8cyAYA](https://mp.weixin.qq.com/s/t1Tta6pdFNqIHkwU8cyAYA)

上周，全球最大的 AI 模型聚合平台 OpenRouter 上，突然冒出了一个匿名模型。

没有署名，没有宣传，没有任何官方背景介绍。代号只有两个字：Hunter Alpha。

![图片](https://mmbiz.qpic.cn/sz_mmbiz_png/JYWVQlsu9znzV2KxUh5fNkFp96jMM7NZ2sIDLbYYlh4Fw5SctWEwVbTC518JJ6uXVWrWU7icvDDE0KibhLyCI7Ewpeu6icooTbtyIDmcAibN4XQ/640?wx_fmt=png&from=appmsg)

然后，这个没有名字的模型，在七天之内，调用量突破了 1 万亿 Token，连续多天霸占日榜第一。全球的开发者圈子都炸了。大家都在猜：这到底是谁家的？

最主流的猜测指向 DeepSeek。理由很充分——模型自称"以中文训练的中国 AI 模型"，知识截止时间是 2025 年 5 月，跟 DeepSeek 现有模型完全一致。再加上万亿参数、100 万 Token 的上下文窗口，所有人都觉得这就是传闻中的 DeepSeek V4 提前泄露了。

![图片](https://mmbiz.qpic.cn/mmbiz_png/JYWVQlsu9zk6smQxZKZoXNt6n7iaodMeljUoPs5poJ0dpydHlJRDPoLC4Zs9C9RkYOmCibJYJqibL1gUopD6u7J2defO2M1cXW9JmFw329SL8A/640?wx_fmt=png&from=appmsg)

连 OpenClaw 的创始人 Peter Steinberger，也就是大家叫的"龙虾之父"，也在 X 上发帖公开打听。

路透社都跟进报道了。一时间全网都在等 DeepSeek 出来确认。

结果 DeepSeek 没回应。

谜底揭晓：做出这个模型的，是小米
----------------

就在昨天，小米官方正式认领了。

Hunter Alpha 和同期上线的另一个匿名模型 Healer Alpha，都是小米 MiMo-V2 系列大模型的内测版本。

而这里有一个非常有意思的背景：罗福莉本人，就是 DeepSeek 的前研究员。

我看到这条信息的时候愣了一下。你想想这个故事线——一个从 DeepSeek 出来的人，在小米做了一个让全网以为是 DeepSeek 的模型。

这事本身就够写一个剧本了。

  

说回正题。小米这次发布的不只是一个模型，而是一整套。

\- MiMo-V2-Pro：文本旗舰，万亿参数，专门给 Agent 当大脑用的

\- MiMo-V2-Omni：全模态模型，能看图、能听声、能操作电脑浏览器

\- MiMo-V2-TTS：语音合成模型，会说方言、会唱歌、还能演戏

用人体来打比方的话：Pro 是大脑，Omni 是眼睛耳朵和手，TTS 是嘴巴。

三个加在一起，就是一个完整的 Agent——能想、能看、能听、能干活、还能开口跟你说话。

![图片](https://mmbiz.qpic.cn/mmbiz_png/JYWVQlsu9zm0miaxFQIThV6hZ7g271faLgWpsOFCduuYiasKd2mNW5rBWdBAz1CfG552KoGpDCEYxcomdu9dE5ZOicrzG8BzTJcQu1eSlMYxf4/640?wx_fmt=png&from=appmsg)

Pro 到底有多强
---------

先说说让全网误以为是 DeepSeek V4 的那个 Pro。

MiMo-V2-Pro 总参数量突破 1 万亿，激活参数 42B。上下文窗口支持到 100 万 Token。

这些数字可能对非技术读者没什么感觉。换个说法：它能一次性"看完"相当于好几本书的内容，然后在这些内容的基础上帮你推理、写代码、做决策。

![图片](https://mmbiz.qpic.cn/sz_mmbiz_png/JYWVQlsu9zmicyCvQxb2AP5Uq4rgC1DEOOGibORhaic7G9xqwzTdS0X7AOPAwGNnQB58OYiaTubJPxIYJvJxcsl422wxibdzVMe0ia1wvH9D0qF4c/640?wx_fmt=png&from=appmsg)

在全球权威大模型综合智能排行榜 Artificial Analysis 上，MiMo-V2-Pro 排到了全球第八，国内第二。

![图片](https://mmbiz.qpic.cn/sz_mmbiz_png/JYWVQlsu9zlKVgpbM8ias9cLzQ0Wnucbnn20GcHmas4QP0fQtZNsNl5Ck1MBichFMZIfw0lUOXH0Gqdk2L3ax7qBuZMDibBgo19ibWiaCzW1AKEg/640?wx_fmt=png&from=appmsg)

但比跑分更有说服力的，是它在 Hunter Alpha 匿名测试期间的实际使用数据。OpenRouter 上调用量最高的几类应用，几乎都是编程相关的开发工具。开发者们在不知道这个模型是谁做的情况下，用脚投了票。

小米内部工程师做过深度对比评测，结论是：MiMo-V2-Pro 的代码工程能力已经接近 Claude Opus 4.6，甚至在系统设计和代码风格上还更优雅一些。

在通用 Agent 能力评测 ClawEval 中，它得了 61.5 分，逼近 Claude Opus 4.6。在 PinchBench 上排名全球前三。

而且这个模型有一个很厉害的特点：它真的能"干活"。

比如有开发者让它一次性生成一个完整的 3D 塔防游戏，包括多种防御塔、不同敌人机制、关卡设计、爆炸火焰特效，基于 Three.js 渲染，还要有暂停、重开、计分功能。它直接给出了结构完整的代码方案。

![图片](https://mmbiz.qpic.cn/mmbiz_png/JYWVQlsu9zlPzJDpJZNpH6lNpC7b4Y87KibY4icTMhtaeGEibvL5qQG9TwpAiadYRzXBsrbOovVeBicZwUtMqWAianR3olwYvt9lYvkfIRtJ2gBq0/640?wx_fmt=png&from=appmsg)

另一个测试更有意思：让它做一个模拟 90 年代纸质杂志风格的网页，不规则多栏排版、出血标题、纸张纹理背景、翻页动效。这种审美层面的理解能力，加上能直接生成前端代码，说明它已经从"生成内容"在往"生成系统"的方向走了。

Omni：能帮你砍价的 AI
--------------

再说 MiMo-V2-Omni，这个更有意思。

它是小米做的全模态模型，能同时处理文字、图片、音频和视频。

但关键不在于"能看能听"，很多模型都能做到这一步。关键在于它能在看完听完之后，直接动手帮你操作。

![图片](https://mmbiz.qpic.cn/mmbiz_png/JYWVQlsu9znJ20ib2OKPPBCmXWPbwUYb9dxzZqnToFJ2kIwaszQvqicQ7aibVX89OEhJAC1xLNhXUu6JvficoNSGWE5YCfapKUdITqhicBGIZP3M/640?wx_fmt=png&from=appmsg)

小米演示了一个场景：Omni 接入 OpenClaw 框架之后，像真人一样操控浏览器——先在社交媒体翻评测，整理出购买建议，然后切到京东跨店比价，联系客服争取优惠，最后完成下单。碰到多标签页切换、实时交互这种复杂操作，都能见招拆招。

说白了，就是你以后可以告诉 AI "帮我买一台性价比最高的洗碗机"，然后它自己去搜、去比、去砍价、去下单。

Healer Alpha 匿名内测的时候，它在 PinchBench 上拿了均分第一。这个成绩的含金量很高，因为 PinchBench 考的就是 Agent 在真实环境里完成任务的能力。

音频理解也很强，支持超过 10 小时的连续长音频，从环境声分类到多说话人分离都能做。综合表现超了 Gemini 3 Pro。

Omni 的 API 也已经开放了，定价是输入 0.4 美元 / 百万 Token，输出 2 美元 / 百万 Token。这个价格放在全模态模型里，属于非常便宜了。

TTS：会打哈欠、会醉酒、还能唱歌
-----------------

第三个模型 MiMo-V2-TTS 是语音合成。

我之前对 AI 语音合成的印象一直停留在"能听清楚但一听就知道是机器"的阶段。但小米这个 TTS 的演示听下来，说实话有点超出预期。

它能做到在同一句话里面完成语气转折和情感变化，比如前半句平静叙述，后半句突然激动起来。这种细粒度控制在同类产品里确实不多见。

方言支持也很全：东北话、四川话、河南话、粤语、台湾腔都有。而且还能唱歌。

小米说未来会把 TTS 和 Omni 深度融合，让 Agent 不光能看懂世界、理解世界，还能用有表现力的声音去"讲述"这个世界。

价格：竞品的五分之一
----------

这次定价是我觉得最值得关注的部分。

MiMo-V2-Pro 的 API 价格：256K 上下文以内，输入 1 美元 / 百万 Token，输出 3 美元 / 百万 Token。1M 上下文范围内，输入 2 美元，输出 6 美元。

这个价格大约是同级别竞品（比如 Claude Opus）的五分之一。

![图片](https://mmbiz.qpic.cn/mmbiz_jpg/JYWVQlsu9zmfAIAxXrtORwKOCYWr8avaZtibsmn2ueNv3S5RocibpjJUiaawibu4CWxfWPuJAgfBbhPYxPnvaDkzBIoz9Sm96l8P1HwZL0AjV0o/640?wx_fmt=jpeg&from=appmsg)

而且小米还搞了一波更狠的操作：联合 OpenClaw、OpenCode、KiloCode、Blackbox、Cline 五大 Agent 框架，限时一周免费调用。

这招很熟悉——先让开发者免费用上，等大家习惯了，再慢慢收费。但对于开发者来说，白嫖一周顶级模型的机会，谁不想试试？

![图片](https://mmbiz.qpic.cn/sz_mmbiz_jpg/JYWVQlsu9zkfl0XABXd6YSMP0KtMhgJCqxokn6vvIoEXBDwuG9WjhUTSYtMEehMP5LyCK3Mb3Jp4tAQfVacojNp3ITMX3VWvWnh3wGYZT0Q/640?wx_fmt=jpeg&from=appmsg)

小米到底在下一盘什么棋
-----------

看到这里你可能会想：小米做手机做车做家电，怎么突然在大模型上这么猛？

其实仔细一想，逻辑很通的。

小米这几年一直在推"人车家全生态"，手机、汽车（SU7）、智能家居通过澎湃 OS 全部打通了。硬件层面的连接早就做好了，但一直缺一个真正能跨设备理解你意图、自己规划任务、自己去执行的智能中枢。

现在，Pro 负责想，Omni 负责看和听，TTS 负责说。三个模型加在一起，就是给整个小米生态装了一套完整的大脑。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/JYWVQlsu9zlJUsHEasBdLiaYzouJ7nB11Y4GsPpEZuiaA9aUM8QXHEB7fUNUGrsZIJNhAnhhric9a48GdbcFkMCXlXyyTbibicQI9D06UWgxqdIk/640?wx_fmt=png&from=appmsg)

miclaw（小米手机端的 AI 智能体）已经接入了 MiMo 大模型。WPS 灵犀也接入了，能直接生成 Word、Excel、PPT、PDF。小米浏览器也在接入中。

罗福莉自己说了一句很有意思的话：从聊天范式到 Agent 范式的转变，快到连她自己都难以置信。她甚至给团队下过一道非常狠的指令——"明天对话次数少于 100 次的成员可以辞职"。

这话听着极端，但背后的信号很明确：小米在 Agent 这条路上，已经下了死命令。

跟普通人有什么关系
---------

说了这么多技术，回到最现实的问题：这事跟你我有什么关系？

我觉得至少有三个层面的影响：

第一，AI 模型的价格战已经打到了一个新阶段。以前同级别的模型一个月调用费可能要好几百美元，现在小米直接砍到五分之一，还限时免费。这意味着更多小团队、个人开发者都用得起顶级模型了。

第二，如果你是小米用户，未来你手机上、车上、家里的智能设备，很可能会比你想象的更"聪明"。因为背后跑的已经是这套万亿参数的 Agent 模型了。

第三，这也是中国 AI 公司在全球市场上的一次很漂亮的亮相。一个匿名模型，没有任何品牌光环加持，纯靠性能让全球开发者自发使用、自发猜测、自发讨论。这种"先打脸再亮牌"的操作，比任何发布会都有说服力。

最后说几句
-----

我个人最感慨的一点是：半年前聊起中国大模型，大家第一反应还是 DeepSeek、百度文心、通义千问。小米在这个名单里基本排不上号。

结果现在，小米做了一个让全网误认为是 DeepSeek 下一代的模型。

这个故事本身就说明了一件事：AI 这个赛道，格局远没有固定下来。谁都有可能在某个拐点上突然冲出来。

对于我们普通人来说，不需要去猜下一个冠军是谁。更重要的是：当这些越来越强、越来越便宜的 AI 工具摆在你面前的时候，你准备怎么用它。
