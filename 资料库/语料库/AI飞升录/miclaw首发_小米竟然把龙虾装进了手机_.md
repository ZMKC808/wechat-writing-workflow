miclaw首发：小米竟然把龙虾装进了手机！
======================

原创 阿竹 AI飞升录 2026-03-09 19:41 浙江

> 原文地址: [https://mp.weixin.qq.com/s/FK3OjSlRB4a5W7cMVuzTYA](https://mp.weixin.qq.com/s/FK3OjSlRB4a5W7cMVuzTYA)

前几天有个读者在后台问，能不能把龙虾直接装到手机上，平时很少用电脑。这话挺有代表性。大多数人一天到晚摸的是手机，Agent 真要成为下一代交互形态，迟早得往移动端跑。

MWC 上卢伟冰跟 CNBC 说了句：当我们的车进国际市场，你会看到 AI Agent 一起跟着来。当时听着像画饼，结果没几天，小米直接把龙虾搬进了手机。

![图片](https://mmbiz.qpic.cn/mmbiz_png/JYWVQlsu9zmMexY5EZ3Gur1gANOmoNFjJLGcSmKIQlR5vPCClPDsUJqcpJ4aAUsYibnWLGYNXgdv2GyI3l8B3u8NuTfeY4lTyr8yViagnx3gs/640?wx_fmt=png&from=appmsg)

Xiaomi Miclaw 官宣，雷军四个字：手机龙虾。![图片](https://mmbiz.qpic.cn/sz_mmbiz_png/JYWVQlsu9zkZHPg7lViakHguvwWswJyRjYe3dASNzOMWPtjRGDPGy0dibrGd8IKu9HhRNOd5Bib1hftHD54cju0mcZTzt9Tf1qFNy2QJStJJyY/640?wx_fmt=png&from=appmsg)

你不再点 App，你只说意图
--------------

以前用手机，你是接线员。日历、短信、闹钟、天气、导航，各干各的，你得自己记、自己切、自己拼。打开四五个 App，点这里点那里，把一堆功能串起来。

Miclaw 刚好把这事反过来。你只说一句周五早上出差帮我准备一下，它自己去查日程、查天气、设闹钟、出发前提醒。以前你得手动操作一整套流程，现在一句话完事。表面是多了一个语音助手，其实是人调度应用变成 AI 调度服务。

![图片](https://mmbiz.qpic.cn/mmbiz_jpg/JYWVQlsu9zm3iaSv46vcosh7THlx6A9V8OqBiaDau2X1AIHzfF7mfsLHSHoVVdHD8PFZPsHLOERyNaMO0M6vQib5BYugcWvPH2BTs1ia5Sh3fKs/640?wx_fmt=jpeg&from=appmsg)

你提目标，它负责拆任务、选工具、按顺序执行。界面上会实时显示调用了哪个工具、执行到哪一步，透明的，你心里有数。

为什么只有小米能做？
----------

电脑上的 OpenClaw 再强，也动不了你手机里的短信、通讯录、日历。手机厂商不可能给第三方开那种权限。硬做的话，只能走豆包那套：视觉 Agent，模拟点击屏幕。通用是通用，但慢，成功率、稳定性跟编程式 Agent 没法比，而且动了各家蛋糕，当时反应才那么大。

![图片](https://mmbiz.qpic.cn/sz_mmbiz_png/JYWVQlsu9zk5a4Tt58iav8v4qrOrvToDQev9p6GpFQuejqktewL56XAXRFiawDFGAQ9Bt3OjPFGaXcyj81f4wrwHgW56xE2teB4oF1NaRxaws/640?wx_fmt=png&from=appmsg)

小米自己下场就不一样了。Miclaw 以系统应用身份运行，直接越过沙盒，50 多种系统级工具封装好了给它用。读短信、建日程、调文件，都走系统 API，不靠模拟点击。所以它能十几秒总结一堆短信，能一句话批量拒收营销信息。第三方 Agent 做不了这些事，能力够，权限不够。

![图片](https://mmbiz.qpic.cn/sz_mmbiz_jpg/JYWVQlsu9zmDbXCiaBDgTMywPNsZApgaUGvvyvtiaStibgh7BrhsTchWstVj0M9nk7m6NcgP5HvYxPwm0G59EiakDrevdb2Dv42tOyyiclCK8icbY/640?wx_fmt=jpeg&from=appmsg)

更狠的是个人上下文。普通手机收到银行扣费短信，顶多弹个通知。Miclaw 会翻你近三个月短信，发现你在两家内容重合的视频平台同时包月，月底给你一份简报：建议退订其中一个，每年省近 400，附带一键退订。这种能力，没系统权限根本做不到。

米家把护城河挖深了
---------

小米全球连了 10 亿多台智能设备。Miclaw 内置完整米家客户端，空调、灯、窗帘、扫地机器人、门锁，一句话能控。![图片](https://mmbiz.qpic.cn/sz_mmbiz_jpg/JYWVQlsu9znAnVQZovgre2Zia14sic2fS8pQAfGtTQCaQalWF3s3cCNs2Dg1HvhUMusddKubXS6MiaBJ1XrPawFvl6mibxzLaVyyK61Cns7BR78/640?wx_fmt=jpeg&from=appmsg)

以前玩智能家居，得在 App 里写一堆规则，如果晚上 10 点且有人经过传感器，那么开廊灯。这种 IF-THEN 逻辑，本质是程序员思维外包。现在日历上写着 10 点重要客户会议，Miclaw 看到后，手机自动静音，扫地机器人原地待命，来电分级，老板的震一下，快递的直接静默。它结合了你的日程、习惯和全屋设备状态，一句话搞定。

![图片](https://mmbiz.qpic.cn/sz_mmbiz_png/JYWVQlsu9zmu4vN5iamnfLnNtKtITaae29ugcP6BsZibm9RSMO5PDNnFuFvrjp1NJib18XUTtkTaPJdm9u3YPYbQxgRc3qraMVkiaUHMG9G5ezw/640?wx_fmt=png&from=appmsg)

小爱也能控米家，但小爱和 Miclaw 的差别，大概就像 Siri 和 OpenClaw。能用人话控制是一回事，能理解复杂意图、跨应用跨设备串流程，是另一回事。

![图片](https://mmbiz.qpic.cn/sz_mmbiz_jpg/JYWVQlsu9zmFCSVqDFSt8KjfLbZnEfTtib2udv6vKMyze2lagYJ0qVCKVxmJCqSH0KAbSPyRnPRrj8ibf8VuHuKmsbIdRCPFKvzggic6Rt1NSo/640?wx_fmt=jpeg&from=appmsg)

部署零门槛，但还在摸石头
------------

OpenClaw 部署要配环境、装依赖、申请 API，对普通人门槛不低。Miclaw 是系统更新完、下载好，打开即用。不需要你会写代码，不需要懂 MCP。首月免费，正常用没限额。

![图片](https://mmbiz.qpic.cn/mmbiz_png/JYWVQlsu9zkPZAhXQHPz4VWqRZlicOrWT8f6YpibbibOxomNMqiaQ6lzvHTKkuiaW9sLsdRwCs2SucYSWYINhQBtFiaLCppXeHwf38UAqV3Uh9sa0/640?wx_fmt=png&from=appmsg)

目前只支持小米 17 系列，小范围封测，邀请制。稳定性、功耗、复杂场景成功率都还在优化。但方向已经摆在那儿了，语音助手时代你喊设闹钟，大模型时代你让改改这段话，Agent 时代你直接说把这件事从头到尾搞定。用户从提问变成提目标，AI 从回答变成把事情跑完。

云厂商在打什么算盘？
----------

腾讯、阿里为啥拼命推 OpenClaw？一键部署、养龙虾教程满天飞。你部署了，得买云服务器吧？得消耗模型 Token 吧？流量进来，后端算力持续变现。这跟当年小灵通一个逻辑。

Miclaw 走的是另一条路。端侧跑 MiMo 大模型，309B 总参数、15B 激活的 MoE 架构，专门为 Agent 场景优化，推理速度在移动端算独一档。大量计算留在本地，隐私不出去，用户也不用为 Token 持续买单。AI 手机的竞争，早就从比谁会聊，变成比谁能干、谁更省了。

![图片](https://mmbiz.qpic.cn/mmbiz_png/JYWVQlsu9znL5iaj8Vjj1Nfkcdea3WicSNkCic3xeASWMzzpfHqZhnnjHpk2IvRjSFQa8bQQxCm89GqWicmM2j7aAHic6hqtDkcJFyDYZwHvasc8/640?wx_fmt=png&from=appmsg)

MiMo 负责人罗福莉，从 DeepSeek 过来的，之前主导 DeepSeek-V2。她加盟小米时写了一句：智能终将从语言走向物理世界。这话跟小米的人车家全生态对上了。

说到底，Miclaw 跟小爱升级版、OpenClaw 手机移植版都不太一样。它是第一个能在手机上跑的、能调用系统级能力的、有安全保障的 Agent 产品。点 App 变成说意图，交互逻辑在变。硬件厂商下场，生态壁垒摆在那儿。云厂商那套变现逻辑在这条路上走不通。手机里的龙虾，这场仗才刚开始。
