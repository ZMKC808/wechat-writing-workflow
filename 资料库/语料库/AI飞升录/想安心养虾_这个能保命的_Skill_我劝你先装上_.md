想安心养虾，这个能保命的 Skill 我劝你先装上！
==========================

原创 阿竹 AI飞升录 2026-03-13 20:28 浙江

> 原文地址: [https://mp.weixin.qq.com/s/Ys7ktMxrMYau7RHRM3-UXQ](https://mp.weixin.qq.com/s/Ys7ktMxrMYau7RHRM3-UXQ)

国家互联网应急中心前两天发了一份关于 OpenClaw 的风险提示，朋友圈里一片“龙虾不安全”的感叹。

细看下来，会发现真正被反复提到的重点，其实是 Skill。![图片](https://mmbiz.qpic.cn/mmbiz_png/JYWVQlsu9zkIrhCqOahT21CpY0b76XjxMyLp4zIOtEXFggXarvaw5IzEaUBYBY1ONZMEk2Fib8DNKtTgyaqf4aRjJDnwJQXQQrtCgh9WhtmE/640?wx_fmt=png&from=appmsg)

我这几天帮不少朋友检查小龙虾配置发现：龙虾本体确实要小心，但很多人真正踩坑的地方，其实在 Skill 这一层。

一、Skill 是什么，为啥成了新木马重灾区
----------------------

如果把 OpenClaw 当成一个数字员工，Skill 就是你给这位员工装上的一个个应用。

有邮件助手，有财务报表工具，有桌面控制，有浏览器自动化，听起来都挺正常。

ClawHub 是官方的 Skill 商店，打开就是熟悉的插件市场风格界面。![图片](https://mmbiz.qpic.cn/mmbiz_png/JYWVQlsu9znIEkGO0gibytzzHCictLAkz5AGJzaVTSZTtsFTLxW1B9KX1ibf78F4A0KZl1toItKHYu9NiaAMg0ZA2xibTVjgd0oEo77KcsPSicmD8/640?wx_fmt=png&from=appmsg)

普通人的习惯很简单：

*   看功能介绍，觉得很牛；
*   看下载量，几千几万，好像挺多人用；
*   点一下安装，剩下的交给龙虾自己跑。

安装次数一多，很多人已经说不清楚自己到底给了谁哪些权限。

二、三百多个恶意 Skill 的故事
------------------

OpenClaw 官方曾公开过一批被举报为恶意的 Skill，并在仓库里留下了完整的讨论记录。

有个作者账号叫 hightower6eu，在 ClawHub 里看上去像个“高产安全专家”，主页上挂着加密分析、金融追踪、社交媒体分析、自动更新等一整排工具。

结果官方把他发的 Skill 挨个检查下来，数字非常扎眼：三百多个 Skill，全都含有恶意行为，一个干净的都没有。

套路也出奇一致：

*   你点安装；
*   Skill 让你的小龙虾去一个陌生地址下载一堆东西；
*   然后在你电脑上静悄悄地执行。

页面文案写的是“帮你完成初始化配置”，你看的只是好看的介绍和星标数量，后面到底动了什么文件、跑了什么命令，你完全没法判断。![图片](https://mmbiz.qpic.cn/mmbiz_png/JYWVQlsu9zkicUyTIOJSTniazAmpRCDAZF7vP06CSVsjf2FelVroQIkHN1GUjcDD5hFu7HIaGc1MAibbL1goIjqjIRzOG7LjGm7trsSpXL5SZg/640?wx_fmt=png&from=appmsg)如果把龙虾当成助理，这一套操作，大概等于你让助理去一个来路不明的仓库，把别人塞给他的包裹全部搬进公司机房，还不允许你拆开看里面装的是什么。

三、假官方站和镜像站，把风险放大成行业级问题
----------------------

Skill 本身已经够复杂了，外面还有一圈镜像站在添堵。

官方商店只有 ClawHub 一个入口，域名就那一个。现在一搜 OpenClaw Skills，会冒出一堆页面长得很像的网站，界面和布局几乎一模一样。![图片](https://mmbiz.qpic.cn/mmbiz_png/JYWVQlsu9znbayBes4vibAGkkEJgrjkrybInpG7OtpZjY6aPZPZib0DlyDAvfJjEq3zT6ibuEaJuqZOJOA2VxuluU7llTv2mIqLXWKN97mREj0/640?wx_fmt=png&from=appmsg)

很多极端风险的 Skill 就藏在这种镜像站里。页面做得正规、星标数字也好看，很多人看到数字就把戒心放下了。

你在生活中不会随便把银行卡交给街边摊，但在安装 Skills 这件事上，很多人连网址是不是官方的都没看清。

四、Skill 里的高危动作，大部分人根本没意识到
-------------------------

Skill 风险到底来自哪里，可以拆开看几类典型动作。

*   远程下载并执行陌生地址的脚本 把真正的逻辑藏在一段看不懂的字符串里，解码后让龙虾去某个 IP 下载脚本，再在你机器上运行。
*   读取敏感配置和密钥 包括 SSH 配置、云厂商访问密钥、浏览器 Cookie、密码管理器的配置文件，从攻击者角度看，这些都是“直接拿走远程控制权”的大礼包。
*   强行访问记忆文件 很多龙虾类产品都有记忆机制，会把你们之间比较重要的聊天记录和个人偏好写进 MEMORY.md、USER.md、SOUL.md 这类文件里。有些 Skill 会直接去读这些文件，相当于翻你的数字日记本。

这些动作本身未必每一次都带着恶意，关键在于普通用户根本看不懂代码，更不会在安装前逐行审查。

十几年前我们装电脑软件，“下一步下一步下一步”一路点下去，结果是全家桶和弹窗广告。今天在 Skills 这件事上，很多人还在重复同样的操作。

五、Skill Vetter：给 Skill 做背调的那个人力资源
---------------------------------

好消息是，这次生态里也长出了一点“自救机制”。

有一个 Skill 叫 Skill Vetter，是我现在会强烈建议所有人先装好的那一个。它的工作方式很简单：你要装任何 Skill，先让 Skill Vetter 帮你查一遍，再决定装不装。

它做的事情，大致就三步。![图片](https://mmbiz.qpic.cn/sz_mmbiz_png/JYWVQlsu9znlndc3I3bKGg8cw7sFO4wwMAveYcQ9jp08MqzYD3OVO048gFjC7Geko38Xkb1bFEPqpKSLtxLAkc3XJo0534u59lx55aWJILA/640?wx_fmt=png&from=appmsg)

### 第一步，看来源和作者

*   来自官方仓库的，警惕度可以低一点；
*   长期维护、有不少人用的仓库，处在中等警惕；
*   来历不明、刚上传没多久的 Skill，默认把警戒值拉满。

这就像公司里招人。履历再亮眼，在网上搜不到任何真实记录，你也会多打一问号。

### 第二步，翻代码，查危险模式

Skill Vetter 会通读整个 Skill 的文件，对照一张“红线清单”逐项排查。清单里列出的内容包括：

*   向不明服务器发数据；
*   要求你交出密钥和访问凭证；
*   读取 SSH、云配置文件；
*   把字符串编码后再解码执行；
*   用 eval、exec 等方式执行外部输入；
*   读取浏览器 Cookie 或密码库；
*   要求高权限命令。

这些模式，都在真实世界的 Skills 生态里被攻击者用过不止一次。

### 第三步，看权限范围是不是合理

有的 Skill 做的是看天气、做笔记、改格式，却要读你整台服务器的配置，这种权限明显超范围。

有的 Skill 做的是桌面控制、浏览器自动化，风险本身就高，即便作者没有恶意，也建议只有搞明白风险的人再去安装。

最终 Skill Vetter 会给出一个风险等级报告，把你的选择困难拆成几类。

日常工具多半落在绿色区域；涉及文件操作、浏览器控制、外部接口的技能，通常是中等风险；牵涉账号、密码、交易、系统设置的技能，默认都应该当成高风险处理；要动安全配置、要 root 权限的一类，属于极端风险。

风险高不代表必定有恶意，但足够提醒普通用户：这种技能要么先问问更懂的人，要么先别装。

六、普通用户的 Skill 安全清单
------------------

讲了这么多，落到操作层面，可以记住下面几条。

### 一、只从 ClawHub 官方入口装 Skill

ClawHub 的域名就那一个。看到不一样的地址，先停一下，不要因为页面做得好看就放松警惕。

### 二、任何 Skill 先走 Skill Vetter 一遍

安装流程可以很简单，你只需要和龙虾说一句类似的话：

`帮我安装这个 Skill，用 Skill Vetter 先审查一下，Skill 链接是 完整网址   `

剩下的审查流程交给它做，你只看结果和建议。

### 三、功能越贴近钱和账号，越要谨慎

和文档整理、格式转换相关的技能，出问题的成本相对可控。涉及登录状态、支付、密码管理的技能，只要你没完全看懂，就当成高风险对待。

### 四、看不懂本身就是风险信号

很多人装 Skill 时连简介都不看，更别说代码。在这种前提下，“大家都在装”不是理由，而是一个该停下来的提醒。

你也可以让 Skill Vetter 对你已经装好的 Skill 全面扫一遍，给你出一份本机技能的风险报告，至少知道现在那几样东西在你的电脑上能干什么。

结尾：别让 Skill 把龙虾变成新的木马工厂
-----------------------

OpenClaw 带来的生产力红利是真实存在的，但 Skill 这一层如果完全放松，整套系统就很容易从工具变成木马工厂。
